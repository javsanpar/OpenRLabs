INTRODUCCIÓN


Aplicación Open Source, que utiliza la funcionalidad ofrecida 
por opengnsys a traves de su API REST para el acceso a escritorio 
remoto mediante un navegador web.

PRE-REQUISITOS

Tener instalada una versión previa de OpenRLabs.

El actualizador ha sido testado en Ubuntu Server 18.04

ACTUALIZACION

Este paquete contiene solamente las fuentes de la aplicación.
NO CONTIENE ninguna base de datos, por lo que los datos de la 
aplicación ya instalada no se modifican.

Se creara una copia de seguridad web2py.old.tar.gz en el directorio
actual de la aplicación.

Para actualizar:

chmod +x updater.sh
sudo foo updater.sh
